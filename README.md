# hrishikos

**Information:**
 - an operating system written with x86 assembly and C
 - made using guidance from [Operating Systems: From 0 to 1](https://raw.githubusercontent.com/tuhdo/os01/master/Operating_Systems_From_0_to_1.pdf)
<br></br>

**Build/Run Instructions:**
 - clone the repository; there is already pre-built a disk.img located in the build folder
 - to clean the build, enter the command `make clean`
 - to build, enter the command `make`
 - to run, install QEMU and enter the command `make qemu-no-debug`
<br></br>
